### Installation

You can install in development mode using `pip install -e .`

#### Requirements

##### Python
- Python >= 3.4

##### Core requirements
- numpy >= 1.14.3
- scipy >= 1.1.0
- pytest >= 3.7.3
- tqdm >= 4.26.0
- h5py >= 2.8.0
- packaging >= 17.1

##### Compatability
- deprecation >= 2.0.6
- typing >= 3.6.6

##### Coverage
- coverage >= 4.5.2
- codecov >= 2.0.15
- pytest-cov >= 2.6.1

##### Tensorflow
- tensorflow >= 1.12

### About the Name

Keras is Greek for 'horn', and rhinoceroses have horns, thus the name, rinokeras
