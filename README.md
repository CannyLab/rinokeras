### Documentation

Read our docs: https://rinokeras.readthedocs.io/en/latest/index.html

### Installation

You can install in development mode using `pip install -e .`

**Requirements**
- Numpy >= 1.14.1
- Tensorflow >= 1.9.0
- Pytest >= 3.7.0

### About the Name

Keras is Greek for 'horn', and rhinoceroses have horns, thus the name, rinokeras
